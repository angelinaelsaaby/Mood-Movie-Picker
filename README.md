Mood Movie Picker

A Python-based desktop application that recommends movies based on the user's current mood and preferred language.

Developed as a 1st Year B.Tech Computer Science Engineering Mini Project, this application demonstrates the use of Python fundamentals, GUI development using Tkinter, and simple recommendation logic.

---

## 📌 Project Overview

Choosing a movie can be difficult when there are thousands of options available.

Mood Movie Picker solves this problem by allowing users to select:

- 🌍 Preferred Language
- 😊 Current Mood

The application then recommends a random movie matching the selected mood.
